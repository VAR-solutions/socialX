# socialx-frontend

## Project setup

```
npm install
```

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Features

- Authentication using JWT
- Post Image with caption
- Follow and Unfollow users
- Personalised feed of following
- Update profile
- 1-on-1 chat
